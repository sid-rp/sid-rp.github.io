Projects

Hackathon Winner

Kube SRE Gym

Awarded 1st place out of 100+ teams at the OpenEnv Hackathon (PyTorch, Cerebral Valley; $100K+ prize pool). An RL environment where a small language model diagnoses and fixes live Kubernetes incidents on GKE via kubectl, with zero hardcoded knowledge of the cluster. An adversarial incident designer creates targeted scenarios based on tracked weaknesses, with curriculum control and an LLM judge scoring SRE workflow quality across expert personas. Post-trained Qwen-series models with GRPO via TRL + vLLM on H100. The agent learned to discover cluster topology, identify faults, and apply correct fixes from reward signal alone.

Reinforcement Learning Kubernetes GRPO vLLM GKE Claude API

Point Tracking in Egocentric Videos

Explored how self-supervised vision backbones (DINOv2, CroCoV2) transfer to the task of tracking points in egocentric videos. Swapped out the default encoder in a point tracking pipeline, trained on EgoPoints, and compared tracking accuracy across 8, 16, and 24 pixel thresholds.

PyTorch DINOv2 Computer Vision
GitHub Presentation Report

Fine-Tuning Medical QA Models

Fine-tuned Meta LLaMA 3.2 3B for medical question answering using LoRA, QLoRA, and GaLore. Compared parameter-efficient methods head-to-head on USMLE Steps 1-3 and MedQuAD. Best results: 45% USMLE accuracy (GaLore) and 70% MedQuAD F1 (QLoRA), showing that efficient fine-tuning can meaningfully improve domain-specific performance on a 3B model.

LLaMA LoRA / QLoRA NLP Hugging Face
